Galen Rupp Tribute Page


A simple, static webpage with information on the long distance runner Galen Rupp. The webpage was built using only HTML and CSS code. Bootstrap 3 was used to provide responsiveness to the webpage, to make the content adapt to any page. A front end development assignment for freeCodeCamp

Mobile Tic-Tac-Toe Game


Using plain HTML, CSS, and Javscript, a simple two-player Tic-Tac-Toe game for mobile was created. Simple CSS atributes were used to allow the game to adapt to any screen size. Although there is no computer player yet, stay tuned for a single player mode!

Non-Linear Region Recognition


Javascript was used to program a basic Artificial Neural Network that utilized stochastic gradient descent to attempt to recognize the border between two distinct regions. Although it most certainly could use some improvements, it is an interesting demonstration of how eas it is to implement basic machine learning algorithims

Implementing the minimx algorithim with javascript, a simple, yet powerful Connect 4 AI was built. Capable of defeating human players in more than 90% of games, it gained great popularity on Khan Academy, where the proram was built
